#### Date, 2026/04/14

## Description

FlowWise is a full-stack financial management web application that integrates **M-Pesa payments** with personal and business finance tracking.
It allows users to send payments via STK Push, receive payments through **Till (Buy Goods)** and **Paybill**, and automatically record all transactions in a centralized system.

The platform also includes budgeting tools, enabling users to monitor spending, track income, and gain insights into their financial behavior in real time.

---

## Installation

To access this project locally, clone the repository from GitHub and set up both the backend and frontend environments.

---

## Installation Requirements

* Git
* Node.js & npm
* Python (3.8+)
* PostgreSQL
* M-Pesa Daraja API credentials (for payment testing)

---

## Installation Instructions

### Clone the repository

```
git clone https://github.com/your-username/flowwise.git
cd flowwise
```

---

### Backend Setup (Flask)

```
pip install -r requirements.txt
python run.py
```

---

### Frontend Setup (React)

```
npm install
npm run dev
```

## Technologies Used

* **Frontend:** React, TypeScript, Tailwind CSS
* **Backend:** Flask, SQLAlchemy
* **Database:** PostgreSQL
* **Payments:** Safaricom M-Pesa (Daraja API)
* **Authentication:** JWT (JSON Web Tokens)
* **Version Control:** Git & GitHub

## Features

* 📲 STK Push payment requests
* 🏪 Till (Buy Goods) payments for businesses
* 🧾 Paybill payments with account references
* 📊 Automatic transaction recording
* 💰 Budget tracking and spending insights
* 🔐 Secure user authentication
* 🔄 Real-time transaction updates (via polling)
