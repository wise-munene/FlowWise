# FlowWise

A web application to help small and medium enterprises (SMEs) track income, expenses, budgets, and generate financial reports. Built with Flask, React, PostgreSQL, and Python.

# Features

-User authentication (signup, login, logout)
-Add, edit, delete income and expense transactions
-Categorize transactions (Sales, Rent, Salaries, Utilities, etc.)
-Dashboard showing total balance, income vs. expenses charts
-Set budgets per category and get alerts when exceeded
-Download financial reports (CSV/PDF)

# Tech Stack

-Frontend: React.js
-Styling/UI: Tailwind CSS / Material-UI
-Backend: Flask
-Authentication: Flask-JWT / Flask-Login
-Database: PostgreSQL
-Charts: Chart.js / Recharts
